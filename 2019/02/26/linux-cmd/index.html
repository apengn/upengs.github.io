chmod
u 代表用户.
g 代表用户组.
o 代表其他.
a 代表所有.

这意味着chmod u+x somefile 只授予这个文件的所属者执行的权限
而 chmod +x somefile 和 chmod a+x somefile 是一样的

ESC + b– 向左移动一个单词，MAC下建议用ALT + ←
